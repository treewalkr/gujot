## Agent skills

### Issue tracker

Issues live as GitHub issues (uses the `gh` CLI). External PRs are **not** a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical roles plus two extra states (`ready-for-code-review`, `ready-for-fix`). See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context (monorepo): `CONTEXT-MAP.md` at the root points to per-context `CONTEXT.md` files. See `docs/agents/domain.md`.
